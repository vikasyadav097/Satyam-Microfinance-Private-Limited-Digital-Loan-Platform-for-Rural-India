# Satyam Microfinance - Digital Loan Platform

**Satyam Microfinance Private Limited** is a modern, trust-driven financial services platform designed specifically for rural India. This digital application enables users to explore loan options, check their eligibility, and apply for financial assistance with minimal documentation.

## 🚀 Key Features

- **Empowering Homepage**: Instantly communicates value with clear calls-to-action for Business, Home, and Agriculture loans.
- **Digital Loan Application**: A streamlined, step-by-step application process designed for simplicity and transparency.
- **CIBIL Score Check**: Integrated service for users to check their creditworthiness (₹399 per report).
- **Rural Focus**: Tailored messaging and imagery that connects with rural entrepreneurs and families.
- **Clean Minimalism Design**: A professional aesthetic using Deep Blue and Emerald accents, focusing on readability and trust.
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile viewing to ensure accessibility in all regions.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (formally Framer Motion)
- **Icons**: Lucide React

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/satyam-microfinance.git
   cd satyam-microfinance
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:3000`.

## 📁 Project Structure

- `src/components/`: Reusable UI components (Hero, Features, ApplicationForm, etc.).
- `src/App.tsx`: Main application entry and routing logic.
- `src/index.css`: Global styles and Tailwind UI theme definitions.
- `src/constants.ts`: Centralized configuration for loan types and company details.
